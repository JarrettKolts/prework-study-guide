CSS

  • Enter CSS Notes here:
  • There are 3 ways to style in CSS:
  • Inline CSS: Styling the webpage within the body of the HTML file
  • Internal CSS Style Sheet: Styling withing the Head of the HTML file
  • External CSS Style Sheet: Links a CSS file to the head of the HTML file
  • The * indicator is a selector that selects all visible attributes
  • The margin affects the spacing outside of the attribute
  • The padding affects the spacing inside of the attribute
  • DRY code (Do not repeat yourself) is important to keep code short and simple

Git

  • Enter your Git notes here:
  • Git status: checks the branch you are in
  • Git checkout -b branch-name: creates a branch off your main branch
  • Git checkout: moves you to the main branch
  • Git add: adds a file to your commit stage
  • Git add -A: commits all files to the working branch
  • Git commit -m: Commits file and adds a commit message.
  • Git branch: shows a list of your branches with a * next to current branch
